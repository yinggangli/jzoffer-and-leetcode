## JZ5  替换空格 :flushed:
`简单` `题目`<br>
*请实现一个函数，把字符串 s 中的每个空格替换成"%20"。* <br><br>
`考点`<br>
<br>
`代码`<br>
```java
class Solution {
    public String replaceSpace(String s) {
        int n = s.length();
        char[] array = new char[n * 3];
        int index = 0;
        for(int i=0; i<n; i++){
            char c = s.charAt(i);
            if(c == ' '){
                array[index ++] = '%';
                array[index ++] = '2';
                array[index ++] = '0';
            } else{
                array[index ++] = c;
            }
        }
        String res = new String(array, 0, index);
        return res;
    }
}
```
